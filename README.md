# Portfolio de Rodi Brian Embony

## Structure
- Le site est une landing page statique en HTML/CSS/JS.
- Les sections principales couvrent le hero, là propos, les compétences, l expérience, les projets, la formation, les certifications, le bénévolat et le contact.

## Images placeholders
Les fichiers de remplacement se trouvent dans le dossier assets/images :
- placeholder-profile.svg
- placeholder-about.svg
- placeholder-ijeeryapp-dashboard.svg
- placeholder-subm.svg

## Remplacer les images
1. Remplacez les fichiers SVG par vos propres images (PNG, JPG ou WebP).
2. Conservez les mêmes noms de fichier pour éviter de modifier le HTML.
3. Si vous souhaitez changer les dimensions, ajustez les styles dans assets/css/style.css.

## Lancer le site localement
Depuis la racine du projet, vous pouvez ouvrir index.html directement ou lancer un petit serveur web :

```bash
python -m http.server 8000
```

Puis ouvrez http://localhost:8000.
